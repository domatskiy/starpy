### Installation
--
##### For Windows you can run:

### `python3 -m venv venv`
### `venv/Scripts/activate`
### `pip3 install -r requirements.txt`

--
##### For Linux you can run:

### `make init`
### `source venv/bin/activate`
### `make install`
